# Email Types by Category

## Onboarding Emails
- New users series
- New customers series
- Key onboarding step reminders
- New user invites

## Retention Emails
- Upgrade to paid
- Upgrade to higher plan
- Ask for review
- Proactive support
- NPS survey
- Referral

## Billing Emails
- Switch to annual
- Failed payment recovery
- Cancellation survey
- Renewal reminders

## Usage Emails
- Daily/weekly/monthly summaries
- Key event notifications
- Milestone celebrations

## Win-Back Emails
- Expired trials
- Cancelled customers

## Campaign Emails
- Monthly roundup
- Seasonal promotions
- Product updates
- Pricing updates

## Email Audit Checklist

For each email in a sequence, verify:

### Content
- [ ] Clear purpose (one job per email)
- [ ] Strong hook (first line grabs attention)
- [ ] Value before ask
- [ ] Single primary CTA
- [ ] Appropriate length for type
- [ ] Conversational tone
- [ ] Active voice

### Technical
- [ ] Subject line (40-60 chars, clear > clever)
- [ ] Preview text (90-140 chars, complements subject)
- [ ] Personalization working (merge fields, fallbacks)
- [ ] Links functional
- [ ] Mobile-responsive
- [ ] Unsubscribe link present

### Sequence Logic
- [ ] Timing appropriate (delays between emails)
- [ ] Exit conditions defined
- [ ] No conflicting sequences
- [ ] Segment conditions correct
- [ ] Goal tracking configured
