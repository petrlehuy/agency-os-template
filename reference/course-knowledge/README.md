# Course Knowledge

Your methodology files go here. Organize by topic — one file per pillar of your service.

Example structure for an email marketing agency:
- `campaigns.md` — campaign strategy, types, timing
- `automations.md` — automation workflows, triggers, best practices
- `copywriting.md` — email copy principles, frameworks
- `deliverability.md` — inbox placement, authentication, reputation
- `list-growth.md` — popup strategy, lead magnets, acquisition

Example for a social media agency:
- `content-strategy.md` — pillar model, calendar planning
- `paid-ads.md` — campaign structure, targeting, budget
- `organic-growth.md` — algorithm, engagement, hashtags
- `analytics.md` — KPIs, reporting, attribution

These files are what Claude loads before generating any strategic output. The better your methodology is documented here, the better Claude's work will be.
