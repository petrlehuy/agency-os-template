# SOPs (Standard Operating Procedures)

Step-by-step procedures for every repeatable task in your agency. Claude loads these on-demand when implementing or explaining a specific task.

Name files descriptively:
- `campaign-setup.md`
- `ab-test-setup.md`
- `client-onboarding-checklist.md`
- `monthly-report-process.md`

The more detailed your SOPs, the more Claude can execute without asking questions.
