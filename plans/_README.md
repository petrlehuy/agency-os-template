# Plans

Brainstorm outputs, strategic plans, and implementation roadmaps. Created during brainstorm sessions with Claude Code, referenced during execution.

## Conventions

**Naming:** `[topic]-[descriptor].md` (e.g., `service-packages-v2.md`, `workflow-automation.md`)

**Status:** Claude manages this automatically — no need to update manually.
- `Draft` — brainstorm in progress or just captured
- `Approved` — AM approved, ready to execute
- `In Progress` — actively being built
- `Done` — completed and implemented

## Template

```markdown
# [Plan Title]
**Status:** Draft

## Key Ideas
- ...

## Next Steps
- [ ] ...
```
